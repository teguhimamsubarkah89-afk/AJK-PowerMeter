# AJK-PowerMeter

IoT Power Meter using ESP32, PZEM-004T, Firebase Realtime Database, and Modbus RTU.

---

This project is a [Next.js](https://nextjs.org) app and includes a production-ready setup for deploying to Firebase Hosting + Cloud Run (container) or other providers.

## Getting Started (development)

Install dependencies and run the development server:

```bash
npm ci
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser.

## Production & Deploy

This repository contains configuration for Docker, GitHub Actions, and Firebase Hosting + Cloud Run. See `FIREBASE_DEPLOY.md` for step-by-step instructions to create a Service Account and set up GitHub Secrets for automated deploys.
